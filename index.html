About Sean Samarov

I'm Sean, a full-stack web developer and Linux system administrator.

I started my journey into tech at the age of 15, when I received a Chromebook laptop for my birthday. Within a few months I'd installed Ubuntu Linux on it and was using it as a gaming machine. Since then, I've continued to pursue all aspects of system administration.

Within a few years, I gained an interest in web development, and began making basic websites for friends and personal projects. This soon became a primary interest of mine, as it satisfied both my technical interest, as well as my more artistic side. This, along with my system administration experience, enable me to offer freelance development, hosting and IT consulting services through my company, Ineptitech.

Project Philosophy

You'll find that this website is completely functional without JavaScript, working entirely with HTML5 and CSS3. You'd be hard-pressed to find a website that isn't written in some kind of MVC framework or templating engine, even if it's just a static page. I stand by the idea that less code is better code, and less design is better design. I believe the web would be a better place if that philosophy was more widespread.
